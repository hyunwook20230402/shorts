## 문제
ComfyUI 로컬 서버(Windows, RTX 4070 Laptop)에서 FLUX.1 Dev fp8 모델로 이미지 생성 시
EmptyFlux2LatentImage에 지정한 해상도의 약 50%로 축소되어 출력됨.

## 원인
FLUX.1 모델 자체의 특성이 아니라 로컬 ComfyUI 환경(버전 0.18.1) 설정의 특성으로 추정.
외부 서버(Linux, RTX 4090)에서는 정상 해상도로 출력되는 것으로 보아 환경/버전 의존성 있음.

## 해결책
원하는 최종 해상도의 **2배**를 EmptyFlux2LatentImage에 입력.

### 예시
**원하는 최종 해상도:** 512×912 (9:16 웹툰 형식)
**실제 입력 설정:** 1024×1824
**실제 출력 결과:** 512×912 ✅

## QC 검수 시 주의사항
- 해상도 확인을 최우선 항목으로 배치 (기술적 품질 검수 첫 단계)
- 합격 기준: 512×912 (정확히 또는 4픽셀 이내 오차 허용)
- 불합격 패턴: 256×456 (1차 시도, 보정 전) → 재생성 요청 필수
