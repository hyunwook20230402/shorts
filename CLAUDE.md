# 프로젝트 개요
고전 시가 → 웹툰 쇼츠 자동 생성 파이프라인

## 기술 스택
- Python 3.11
- ComfyUI: ~/comfyui (API 모드로 실행 중, port 8188)
- 베이스 모델: Anything V5 (~/comfyui/models/checkpoints/)
- TTS: Edge-TTS (로컬), ElevenLabs (API)
- 영상 합성: FFmpeg + MoviePy

## 코딩 규칙
- 타입힌트 필수 (Python 3.10+ 문법 사용)
- 함수 단위로 파일 저장 — 중간 결과물은 반드시 디스크에 캐시
- API 호출은 retry 3회 + 지수 백오프
- 로그는 print 대신 logging 모듈 사용
- 환경변수는 .env 파일 — 코드에 하드코딩 금지
- 한 함수는 한 가지 일만 (단일 책임 원칙)

## 응답 규칙
- 모르는 정보면 모른다고 말하기
- 코드 수정 시 diff 형태 또는 변경 함수만 출력
- 동작 원리 재설명 없이 바로 구현
- 요청하지 않은 리팩토링 금지
- "더 좋은 방법이 있습니다" 제안은 요청 완료 후 한 줄로만
- 에러 메시지를 받으면 원인 분석 먼저, 수정 코드 나중
